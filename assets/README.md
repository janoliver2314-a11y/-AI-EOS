# Assets

Static, non-code assets (images, exported diagrams, logos, etc.).

## Conventions

- Name files descriptively in `kebab-case`, matching
  `docs/standards/naming-conventions.md`.
- Exported diagrams belong here only if they can't be expressed as inline
  Mermaid; see `docs/diagrams/README.md` for the diagram convention.
- Do not commit large binary assets without considering repository size —
  prefer optimized/compressed formats.
