# Future Ideas

A parking lot for ideas that have been considered but are not prioritized.
Entries here are **not commitments** — see `memory/roadmap.md` for actual
priorities. An idea graduates to the roadmap when it's deliberately
scheduled.

## Ideas

- **Documentation staleness detection** — tooling that flags docs that
  haven't been updated despite the code they describe changing. Related:
  `docs/volumes/01-foundation.md` Future Improvements.
- **Automated doc-skeleton linter** — check new Markdown files under
  `docs/` for the required section headings from
  `docs/standards/documentation.md`.
- **Lesson-entry tagging** — tag `memory/lessons-learned.md` entries by
  domain (security, performance, concurrency) once the file is large
  enough that a flat list is hard to scan.
- **Cross-repo adoption guide** — a guide for applying AI-EOS's structure
  to a different repository, once this one has proven the methodology out.
- **Auto-generated master reference table** — generate the volume index in
  `docs/volumes/10-master-reference.md` from document front matter instead
  of maintaining it by hand.
