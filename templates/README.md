# Templates

Reusable document templates. Copy the relevant template rather than writing
a new document type's structure from scratch each time.

| Template | Use for |
|---|---|
| `adr-template.md` | Architecture Decision Records (`docs/decisions/`) |
| `bug-report-template.md` | Filing a bug (see also `.github/ISSUE_TEMPLATE/`) |
| `feature-spec-template.md` | Proposing a new feature before implementation |
| `pr-description-template.md` | Pull request descriptions |
| `postmortem-template.md` | Incident postmortems |
